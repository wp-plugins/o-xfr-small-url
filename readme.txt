Help your readers to reduce your posts links sizes. It's like TinyUrl with stats.

== Description ==
It's really easy to have extremely tiny url! Simply clic on Activate!

== Installation ==

1. Upload `o-x-badge` unziped directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
